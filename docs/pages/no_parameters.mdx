# Config-only Mode

If you only need to use `Component` objects for configuration purposes,
you can use `no_parameters` context to suppress the creation of any
parameters and only set the `config` attribute.

```python
import torchmix
from torchmix import nn

with torchmix.no_parameters():
    nn.Linear(1000000, 1000000)  # Fast! ⚡ ️
```

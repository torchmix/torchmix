# Transpose (Feedforward)

Applies Feedforward to penultimate dimension.

```python copy
Feedforward(
    dim=196,
    plugins=[
        Transpose(),
    ],
)
```

This plugin can be used to implement token-mixer from [Feedforward-Mixer](https://arxiv.org/abs/2105.01601)
